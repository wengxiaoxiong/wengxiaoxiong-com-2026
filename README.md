# wengxiaoxiong-com-2026

Personal website for Weng Xiaoxiong.

Built as a bilingual pixel-art portfolio for an AI-native startup founder and Agent systems builder. The site emphasizes commercial value, product thinking, agent architecture, and proof-of-work through projects and content.

## Features

- Default English experience with browser-language based Chinese switching.
- Manual EN / ZH toggle.
- Pixel-art visual system with generated hero artwork.
- Founder-oriented sections for business systems, Agent technology, commercial experiments, and Bilibili content proof.
- Static export support for easy deployment.

## Development

```bash
pnpm install
pnpm dev
```

## Build

```bash
pnpm build
```

Static output is generated in `out/`.
